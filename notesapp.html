About the Project

Note Calendar is an Android application built using Jetpack Compose and Room Database. The app allows users to create, update, and manage notes efficiently by linking them with specific calendar dates.

Users can tap on any date in the calendar to instantly view notes associated with that day, making note organization simple and intuitive.

üìù App Features

  • Create, update, and delete notes
  • Calendar-based note filtering
  • Tap on a date to view notes for that day
  • Persistent local storage using Room Database
  • Clean and minimal UI built with Jetpack Compose
  • Date and time display for each note
  • Floating action button for quick note creation
